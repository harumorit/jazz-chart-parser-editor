構文木エディタ v1

Chart Parserが出力した構文木JSONを読み込み，ノードの編集・部分木の置換・コード列の再生成を行うための試作版です。

1. 構文木JSONを読み込む

Parser側の「最終候補木をJSON保存」や「全ステップJSON保存」から出したJSONをここに貼り付けて使う想定です。

2. 構文木ビュー

?IC は未確定ノード，Cmaj7 は実際の入力コード，IC はカテゴリです。クリックしたノードを右側で編集できます。

部分木の置換

    
選択中ノードを，よく使う構造に置き換えます。編曲ルールの試作として使えます。

    
    
    
    
    
置換の意味

    
例：V-I置換は「緊張→解決」の構造，II-V-I置換は「準備→緊張→解決」の構造を木として入れます。

  


  

    
研究上の位置づけ

    
このエディタは，Parserが出力した構文木を編集可能な中間表現として扱うためのものです。今後は，編集後の構文木からコード進行を再生成し，編曲案として利用する流れを目指します。

    
コード進行 → Chart Parser → 構文木 → 構文木エディタ → 編集後コード列 → 編曲案
